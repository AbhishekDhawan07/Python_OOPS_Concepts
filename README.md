# 🐍 Python_OOPS_Concepts

This repository is dedicated to understanding and implementing **Object-Oriented Programming (OOP) concepts in Python**. It contains Jupyter Notebook files that explain OOP principles in a structured and practical way.

---

## 📁 Folder Structure

### 📂 OOPs Concepts in Python

This folder contains the following notebooks:

1. **OOPS Concepts in Python - Part 1.ipynb**  
   - Introduction to OOPS Concepts
   - Classes and Objects
   - Constructors (`__init__`) and it's Types
     
2. **OOPS Concepts in Python - Part 2.ipynb**  
   - Encapsulation
   - Abstraction
   - Inheritance
   - Polymorphism
   

---

## 🚀 Getting Started

To explore the notebooks:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Python_OOPS_Concepts.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Python_OOPS_Concepts
   ```

3. Open Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

4. Navigate to:
   ```
   OOPs Concepts in Python/
   ```
